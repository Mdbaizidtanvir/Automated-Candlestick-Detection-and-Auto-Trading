# Automated Candlestick Detection and Auto Trading

Hi, I’m **MD Baizid Tanvir** —  
Full Stack Developer specialized in **Python, Django, React**, and full-stack product development.  

This project is an automated system for detecting candlestick patterns and executing trades based on technical analysis. It’s designed for **real-time market monitoring** and automated trading strategies.

---

## Features

- **Real-time Candlestick Detection**  
  Detects common candlestick patterns like Doji, Hammer, Engulfing, etc.

- **Automated Trading Signals**  
  Generates Buy/Sell signals based on candlestick patterns and predefined strategies.

- **Integration with Trading Platforms**  
  Compatible with brokers or platforms for automated order execution.

- **Technical Indicators**  
  Supports RSI, EMA, SMA, MACD, Bollinger Bands, Supertrend, and more.

- **CSV & WebSocket Data Input**  
  Can read historical data from CSV and live updates via WebSocket for real-time decision-making.

- **Custom Alerts**  
  Email and console alerts when a trade signal is generated.

---

## Tech Stack

- **Frontend:** React, Next.js, Tailwind CSS  
- **Backend:** Python, Django, FastAPI  
- **Data & Analysis:** Pandas, NumPy, TA-Lib or `ta` library  
- **Automation:** Selenium / WebSocket integration for live market data  
- **Database:** SQLite / PostgreSQL (optional)  
- **Deployment:** Vercel / Heroku / Docker
